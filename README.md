# Информационный web-сайт ФКУ «Главное бюро медико-социальной экспертизы по Нижегородской области»

Учебный проект для производственной эксплуатационной практики по теме:
**«Проектирование и разработка информационного web-сайта для ФКУ “Главное бюро медико-социальной экспертизы по Нижегородской области”»**.

## Что реализовано

- 7 основных страниц сайта:
  - Главная;
  - Об учреждении;
  - Услуги МСЭ;
  - Гражданам;
  - Документы;
  - Новости;
  - Контакты.
- Служебные страницы:
  - карта сайта;
  - страница 404.
- Лента новостей.
- Полезные материалы по категориям.
- Поиск по сайту.
- Форма обратной связи.
- Система сообщений: пользователь отправляет сообщение, администратор просматривает его в панели управления.
- Авторизация пользователей.
- Самостоятельная регистрация пользователя.
- Административная панель.
- Управление новостями.
- Управление полезными материалами.
- Просмотр сообщений.
- Управление пользователями.
- Хлебные крошки.
- Стандартный стиль сайта.
- Версия для слабовидящих.
- Автоматическое создание и заполнение базы данных SQLite при первом запуске.

## Запуск проекта

1. Установить Python 3.10 или новее.
2. Распаковать архив проекта.
3. Открыть терминал в папке проекта.
4. Создать виртуальное окружение:

```bash
python -m venv venv
```

5. Активировать виртуальное окружение:

Windows:

```bash
venv\Scripts\activate
```

macOS / Linux:

```bash
source venv/bin/activate
```

6. Установить зависимости:

```bash
pip install -r requirements.txt
```

7. Запустить сайт:

```bash
python app.py
```

8. Открыть в браузере:

```text
http://127.0.0.1:5000
```

## База данных

Файл базы данных создается автоматически:

```text
instance/site.db
```

При первом запуске создаются таблицы и демонстрационные записи: пользователи, новости, материалы, статические страницы.

## Примечание для отчета

Проект выполнен без CMS. Сайт разработан на Python с использованием микрофреймворка Flask и базы данных SQLite. Для демонстрационного стенда используются тестовые учетные записи. Все элементы интерфейса, меню, формы и сообщения оформлены на русском языке.
